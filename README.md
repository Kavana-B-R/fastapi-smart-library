# 📚 Smart Library Management System API

🚀 A fully functional backend application built using FastAPI to manage a library system.  
This project demonstrates real-world API development including CRUD operations, multi-step workflows, and advanced querying features.

---

## 📌 Project Overview

The Smart Library API allows users to:
- Manage books (add, update, delete)
- Borrow and return books
- Track transaction history
- Search, filter, sort, and paginate books

This project was developed as part of FastAPI internship training.

---

## ⚙️ Features Implemented

### ✅ Core Features
- RESTful API design using FastAPI
- Pydantic data validation
- Proper HTTP status codes
- Error handling (404, 400, 201)

### 📚 CRUD Operations
- Add new books
- Get all books
- Get book by ID
- Update book details
- Delete books

### 🔁 Multi-Step Workflow
- Borrow a book
- Return a book
- Track borrowing history with timestamps

### 🔍 Advanced APIs
- Search books by keyword
- Filter by availability (available / borrowed)
- Sort books by price (ascending / descending)
- Pagination support
- Combined browsing endpoint (search + filter + sort + pagination)
